int padovan ( int n)

Calcola l'n-esimo numero della sequenza di Padovan.

La sequenza è definita come P(n) = P(n-2) + P(n-3) con P(0)=1, P(1)=1, P(2)=1.

Parametri
nL'indice della sequenza.
Restituisce
L'n-esimo numero di Padovan.
